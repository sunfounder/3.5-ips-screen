**インターフェース定義**

以下は3.5インチIPSスクリーンのピン配置図です。実際に接続されているのは最初の26ピンのみで、27～40ピンは未接続です。これらは40ピン仕様のRaspberry Piとの互換性を保ち、誤挿入を防ぐために用意されています。

.. image:: img/3.5_ips_pins.png
  :width: 500
  :align: center

.. list-table:: 
    :header-rows: 1

    * - ピン番号
      - シンボル
      - 説明
    * - 1, 17
      - 3.3V
      - 電源入力（3.3V）
    * - 2, 4
      - 5V
      - 電源入力（5V）
    * - 3, 5, 7, 8, 10, 12, 13, 15, 16, 27～40
      - NC
      - 未接続
    * - 6, 9, 14, 20, 25
      - GND
      - グラウンド
    * - 11
      - TP_IRQ
      - タッチパネル割り込み信号（押下時にLow）
    * - 18
      - LCD_RS
      - LCDレジスタ選択信号（Low：コマンド、High：データ）
    * - 19
      - LCD_SI / TP_SI
      - LCD表示／タッチパネル用SPIデータ入力
    * - 21
      - TP_SO
      - タッチパネルからのSPIデータ出力
    * - 22
      - RST
      - リセット信号（Lowアクティブ）
    * - 23
      - LCD_SCK / TP_SCK
      - LCD表示／タッチパネル用SPIクロック信号
    * - 24
      - LCD_CS
      - LCDチップセレクト信号（Lowアクティブ）
    * - 26
      - TP_CS
      - タッチパネルチップセレクト信号（Lowアクティブ）


**ファン用ピン**

スクリーン背面には外部ファンを接続するための2つのピンがあります。ただし、ファンを接続すると常時回転し、ソフトウェアでの制御はできません。
