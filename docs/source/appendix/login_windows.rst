PuTTY
=========================

Windows ユーザーの場合、SSH クライアントアプリを利用することができます。ここでは `PuTTY <https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html>`_ を推奨します。

**ステップ 1**

PuTTY をダウンロードします。

**ステップ 2**

PuTTY を起動し、左側のツリー状メニューから **セッション（Session）** を選択します。
**ホスト名（または IP アドレス）(Host Name or IP address)** に Raspberry Pi の IP アドレスを入力し、 **ポート（Port）** に **22** （デフォルト値）を入力します。


.. image:: img/image25.png
    :align: center

**ステップ 3**

**開く（Open）** をクリックします。初回ログイン時にはセキュリティ警告が表示されますが、 **はい（Yes）** をクリックしてください。


**ステップ 4**

PuTTY ウィンドウに「 **ログイン名 (login as):** 」と表示されたら、ユーザー名として「 **pi** 」を入力し、続けて **パスワード (password)** として「raspberry」（変更していない場合のデフォルト）を入力します。

.. note::

    パスワード入力時に文字が表示されないのは正常な仕様です。正しく入力してください。  

PuTTY のウィンドウに「非アクティブ（inactive）」と表示された場合、接続が切断されたことを意味し、再接続が必要です。  

.. image:: img/image26.png
    :align: center
