3.5インチ 480×320 IPS タッチスクリーン
==============================================

.. image:: img/3.5_ips_screen.jpg
    :width: 500

この3.5インチ 480×320 IPSタッチスクリーンは、広い視野角、豊かな65Kカラー、高い応答性のタッチ操作を実現します。  
Raspberry Pi向けに設計されており、GPIOに直接接続できるため追加のケーブルは不要です。Raspberry Pi OS、Ubuntu、Kali Linux、RetroPieに対応しています（ドライバーのインストールが必要）。  
DIYプロジェクトやコンパクトな組み込み用途に最適です。

**特長**

* 320×480解像度 – コンパクトなプロジェクトでも鮮明で詳細な表示。
* IPS方式 – 広視野角で鮮やかかつ正確な色表現。
* 高速SPI（最大125MHz、50FPS） – 滑らかで応答性の高い表示性能。
* 抵抗膜式タッチ – 安定した正確なタッチ入力。
* 40ピンGPIOインターフェース – Raspberry Piへ簡単・確実に接続。
* 幅広い互換性 – Raspberry Pi 5、4B、3B/3B+、Zero 2Wに対応。
* 複数OS対応 – Raspberry Pi OS、Ubuntu、Kali Linux、RetroPieに対応（ドライバー付属・インストールが必要）。
* Raspberry Piに最適設計 – Pi本体サイズに合わせたコンパクト設計。

.. note::

   3.5インチIPSスクリーンドライバーをインストール後は、HDMIモニターを取り外すことを推奨します。

   HDMIモニターを接続したままにすると、画面が表示されない、または黒画面になる場合があります。
